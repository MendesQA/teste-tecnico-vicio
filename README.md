# QA Pleno Teste Técnico Vicio de uma Estudante

Projeto de automação de testes utilizando **Cypress** e **Comandos Customizados**.  
Cobre fluxos de login, e publicação de comentários no site [Vicio](https://betaconcursos.metodovde.com.br/entrar).

---
## ⚙️ Dependências

- [Node.js](https://nodejs.org/) ≥ 18
- [Cypress](https://www.cypress.io/) ≥ 13
- npm ou yarn

## ⚡ Instalação

1. Clone o repositório:

```bash
git clone https://github.com/MendesQA/teste-tecnico-vicio.git
cd teste-tecnico-vicio
```
2. Instale as dependências
```bash
npm install
```
3. Instale os browsers do Cypress
```bash
npm install cypress --save-dev
```
4. Execute os testes em modo Headless
```bash
npx cypress run
```

5. Execute os testes em modo Interativo
```bash
npx cypress open
```
